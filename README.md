# Description

Notebooks made for the Computational Musicology 8824 seminar at The Ohio State University's School of Music.

See the syllabus here: [syllabus/Computational_Musicology_8824.02.pdf](syllabus/Computational_Musicology_8824.02.pdf)

The code for this website is hosted here: [https://github.com/fdch/computational_musicology-8824.git](https://github.com/fdch/computational_musicology-8824.git)

# Usage

You can browse the read-only notebooks (html version) nlfollowing) the links in this website ([https://fdch.github.io/computational_musicology-8824](https://fdch.github.io/computational_musicology-8824)), or you can download or clone the repository and run the notebooks:

NOTE: see [requirements.txt](requirements.txt)

```sh
git clone https://github.com/fdch/computational_musicology-8824.git
cd computational_musicology-8824
./config.py -depic
# or simply
jupyter notebook
```
Alternatively, you can browse the notebooks online following its corresponding *colab* link
